How to make responsive

1. The Viewport Meta Tag
<meta name="viewport" content="width=device-width, initial-scale=1.0">

This tells phones: "Don't zoom out, use the actual screen width" - the foundation of all responsive design.

2. Fluid Typography with `clamp()`** ðŸ“

Instead of fixed font sizes, I used:
```css
h1 {
    font-size: clamp(3rem, 8vw, 6rem);
    /* Minimum: 3rem | Preferred: 8% of viewport | Maximum: 6rem */
}

h2 {
    font-size: clamp(2.5rem, 5vw, 4rem);
}
```
This makes text **shrink on phones** and **grow on laptops** automatically.

---

3. CSS Grid with Auto-Fit** ðŸ”²

The project grid automatically rearranges itself:

.work-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(400px, 1fr));
    /* Each column at least 400px wide, but on phones it becomes 1 column */
}

.services-grid {
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    /* Cards stack on phones, side-by-side on laptops */
}
```

**What happens:**
- **Laptop:** 2-3 columns side by side
- **Phone:** Automatically becomes 1 column (stacks vertically)

---

4. Media Queries (The Magic)** ðŸ“²

The most powerful responsive tool - different CSS for different screen sizes:

/* For tablets and smaller laptops */
@media (max-width: 1024px) {
    .container {
        padding: 0 2rem;  /* Less padding on sides */
    }
    
    .about-content {
        grid-template-columns: 1fr;  /* Stacks image and text */
    }
}

/* For phones */
@media (max-width: 768px) {
    .nav-links {
        display: none;  /* Hide desktop menu */
    }
    
    .menu-toggle {
        display: block;  /* Show hamburger menu */
    }
    
    .work-grid {
        grid-template-columns: 1fr;  /* One column for projects */
    }
    
    .stats {
        grid-template-columns: 1fr;  /* Stack stats vertically */
    }
}
```

## **5. Flexible Images** ðŸ–¼ï¸

```css
.work-image {
    width: 100%;        /* Always fit container width */
    height: 500px;       /* Fixed height on desktop */
    object-fit: cover;   /* Crop nicely without stretching */
}

/* On phones, height automatically adjusts */
@media (max-width: 768px) {
    .work-image {
        height: 300px;   /* Smaller on phones */
    }
}
```
